# koishi-plugin-manosaba-text-box
一个基于 Resvg 和 的 wasm-vips 的 Koishi 自动化表情包生成插件，能够快速生成带有自定义文本的魔法少女的魔女裁判文本框图片。

# 用法
```
mtb -c <chara> <text>
mtb <text>
```
